서울 지하철 노선도 관련된 앱을 개발하고 싶거든? 먼저, 서울지하철데이터가 필요해. 데이터를 수집해서 아래 모양으로 json 파일을 만들어줘.

```json
[
    {
        "id": 999,      # unique integer를 어떻게든 부여하면 됨

        # 호선
        # - 숫자가 아닌 경우도 있기 때문에 string type을 사용
        # - 한 역에 여러 호선이 지나갈 때도 있기 때문에 array type을 사용
        "line": ["3", "분당"],

        "name": "수서",  # 역명
        "neighbor": [998, 1000],  # 인접한 역들의 id
    },
    ...
]
```
